タスク管理アプリ「Taskun」

URL

https://taskun-c52638c61c43.herokuapp.com/taskun/

サービスの目的

実家にいる母が、「家事の分担や買い物リストを共有したいけれど、みんなが揃う時間がないからまとめて管理できるものが欲しい」と言っていたのがきっかけです。

使用技術

プログラミング言語: HTML, CSS, Python
フレームワーク: Django

機能概要

リスト・カード機能により、複数のタスクをまとめて管理できます。
また、サインアップ・ログインの認証機能により、誰が作成したかが分かるので、担当や進捗状況の把握もできます。
