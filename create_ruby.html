リアルタイムチャットアプリ「live chat」

URL

https://live-chat-app-a022a20a842b.herokuapp.com/

サービスの目的

学習中のRuby on RailsとVue.jsを実践的に活用するため、実際にアプリを開発しようと考え、日常的に使用しているLINEのようなリアルタイムチャットアプリケーションを作成しました。

使用技術

プログラミング言語: HTML、 CSS、 Ruby、 JavaScript
フレームワーク: Ruby on Rails、 Vue.js

機能概要

メッセージを送信すると同時に他のユーザにメッセージが届くようになっていて、メッセージをダブルクリックすると、いいねをつけることができます。
また、サインアップ・ログインの認証機能があり、ログインしていない場合は、チャットルームにはアクセスできないように制限をかけています。
